# code-auto-refactoring-tool

**Project Overview**

The Code Auto Refactoring Tool is a Python-based application designed to automatically improve the quality and readability of code. It analyzes Python code using Abstract Syntax Tree (AST) and applies rule-based transformations such as variable renaming, formatting correction, and condition simplification without changing the program’s functionality.

**Dataset**

This project does not use a traditional dataset.
Instead, it works on user-provided Python code as input for analysis and refactoring.


**Language Used**

Python
AST (Abstract Syntax Tree module)
